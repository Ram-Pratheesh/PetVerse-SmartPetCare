## 📈 ML Model Info
- Trained using Keras on synthetic dataset (20 breeds).
- Output: Multilabel classification (Health, Grooming, Feeding, Reminder).
- Accuracy:
  - Health: 87.2%
  - Grooming: 78.8%
  - Feeding: 75.0%
  - Reminder: 83.0%
- Exported to `.tflite` and quantized for mobile integration.
